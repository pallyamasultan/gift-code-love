# 💖 Gift Code Love - Animasi Romantis Web

Proyek sederhana berbasis web yang menampilkan animasi partikel interaktif menggunakan HTML5 Canvas. Dibuat khusus untuk memberikan kejutan manis melalui kombinasi efek visual dan musik.

## ✨ Fitur Utama
* **Countdown Interaktif**: Animasi angka 3, 2, 1 yang dinamis sebelum kejutan utama.
* **Latar Belakang Digital**: Efek "Binary Code" berwarna magenta yang padat dan modern.
* **Partikel Hati**: Ribuan partikel yang menyatu membentuk hati yang berdetak (*pulsing heart*).
* **Efek Bintang**: Titik-titik cahaya yang melayang perlahan di latar belakang untuk kesan estetik.
* **Integrasi Musik**: Musik romantis yang otomatis berputar saat tombol diklik.
* **Desain Responsif**: Tampilan tetap rapi baik dibuka di laptop maupun di perangkat seluler (HP).

## 🛠️ Teknologi yang Digunakan
* **HTML5**: Struktur halaman dan elemen Canvas.
* **CSS3**: Penataan gaya, efek Glassmorphism, dan animasi neon.
* **JavaScript (Vanilla)**: Logika partikel, manipulasi canvas, dan kontrol audio.
